# Brain Tumor Detection

AI-powered brain tumor detection and classification platform with a FastAPI backend, TensorFlow deep learning model, and React frontend.

## Features

- MRI image upload and preprocessing
- Multi-class tumor classification (Glioma, Meningioma, Pituitary, No Tumor)
- Confidence scoring for predictions
- Interactive React frontend with result visualization
- REST API for programmatic access
- Model evaluation metrics dashboard

## Tech Stack

Python, FastAPI, TensorFlow, Keras, React, JavaScript

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/brain-tumor-project-main.git
cd brain-tumor-project-main
pip install -r requirements.txt && cd frontend && npm install
```

### Usage

```bash
uvicorn app.main:app --reload
```
